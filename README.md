# Gcriva

Management system for Gcriva, a nonprofit organization that supports children and teenagers.

## Project base
* Based on hackathon-starter boilerplate

## How to run the code

###
1. Run `yarn` (if you don't have yarn, run `npm i -g yarn`)
2. Run `npm start`
